Sales & Customer Insights Dashboard (Power BI)
📌 Project Overview
This project presents an interactive Power BI dashboard built to analyze sales performance, customer behavior, and product trends. The dashboard helps in identifying key business insights for better decision-making.

🎯 Objectives
Analyze sales performance across regions

Identify top-performing product categories

Track sales trends over time

Understand customer distribution

🛠️ Tools Used
Power BI

Excel / CSV Dataset

Data Cleaning & Transformation

📊 Dashboard Preview

🔍 Key Insights
West region generates the highest sales

Technology category contributes the most revenue

Sales show variation across months indicating seasonal trends

Certain products dominate overall sales contribution
